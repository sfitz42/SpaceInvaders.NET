SpaceInvaders.NET

Instructions

Select Space Invaders ROM files

Optional: Select Space Invaders sound files to enable sound

The emulator will automatically start when all ROMs have been uploaded.

Controls

Control Mapped Key
Left Left Arrow
Right Right Arrow
Fire Space
Start 1P Enter
Start 2P Backspace
Add Credit C

Required ROMs

  • invaders.h
  • invaders.g
  • invaders.f
  • invaders.e

Required Sounds

  • UFO (0.wav)
  • Fire (1.wav)
  • Death (2.wav)
  • Hit (3.wav)
  • Fleet Move 1 (4.wav)
  • Fleet Move 2 (5.wav)
  • Fleet Move 3 (6.wav)
  • Fleet Move 4 (7.wav)
  • UFO Hit (8.wav)
